Visor comparativo de cartografía histórica y actual

Visor web desarrollado con HTML, CSS y JavaScript usando OpenLayers que permite explorar y comparar mapas históricos y actuales mediante un deslizador interactivo. Incluye carga de archivos propios, dibujo y medición.

Tecnologías: HTML, CSS, JavaScript, OpenLayers, WMS/WMTS, GeoJSON, PNG

Estado: Puede tener cambios ligeros en las próximas semanas.

Repositorio en GitHub
