Visor Comparador de Cartografía Región

Explora la evolución del territorio a lo largo del tiempo con esta aplicación web interactiva.

A través de un deslizador visual, puedes superponer mapas modernos —como el PNOA, cartografía ráster del IGN y OpenStreetMap— con cartografía histórica, como el vuelo de Ruiz de Alda (1929–1930).

La interfaz ofrece una barra de herramientas intuitiva para añadir, activar o cambiar capas, limpiar el mapa, dibujar geometrías personalizadas y realizar mediciones precisas.

El visor está pensado tanto para profesionales como para cualquier persona interesada en descubrir cómo ha cambiado el paisaje a lo largo del tiempo.
